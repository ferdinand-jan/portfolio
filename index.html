Game Jam 2021 - Constructive Boredom

Un Game Jam de 48 heures que j'ai réalisé seul. Le thème était "Stay Here". J'ai décidé de créer un jeu de type "bac à sable" où le joueur est enfermé dans une pièce pendant 5 heures (5 minutes en réalité). Il n'est donné qu'un seul ordre, qu'une seule indication : on lui dit de "rester ici". Le but pour le joueur est d'essayer de ne pas s'ennuyer pendant ces 5 minutes.
Vous pouvez télécharger le jeu ici

Game Jam 2018 - Julieng the AI

Une Game Jam de 24 heures que j'ai réalisée avec 2 amis. Le thème était "Téléportation". Nous avons réussi à faire un jeu de plateforme composé d'un niveau, où notre personnage ne peut pas sauter. Le joueur doit utiliser des téléporteurs interconnectés pour avancer. J'étais le concepteur du jeu et le programmeur. Notre jeu, "Julieng the AI", a été classé 6e sur les 13 jeux soumis.
Vous pouvez télécharger le jeu ici
